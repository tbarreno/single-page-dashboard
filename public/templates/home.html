Welcome

This is a Rich Internet and Single Page Application base code featuring a management dashboard front-end and REST services with a Mongo database backend.

Application goal

It's a simple TO-DO list management. The lists are under the Projects sidebar link.

Each project has a list of Task (TO-DO) entries with a "done" percentage, a status and some labels.

Architecture

The principal libraries and technologies behind this application are:

  • Node.js with Express as the main server
  • MongoDB as DataBase backend
  • jQuery as the main RIA library
  • EJS as template engine
  • The great Creative Tim's admin dashboard template (Material Dashboard, Bootstrap based) for the user interface
